# Dashboard

## Objective
**To analyze job postings across the globe based on median salary, location and also the job type,whether Part-time or Full-time,as well as the jop platforms and job count**

## Questions answered
 - Which roles have the highest Median Salary?
 - Where are these jobs located?
 - Are they part-time or full-time?
 - On which Job platform are they posted the most?
 - How many times are they posted on those platforms?



*For this project, I had to take raw unfiltered data and learn how to manually clean it up and organise it in a way that would allow for analyses and thus drawing of insights*

**As a first timer and also through following a tutorial, i was a able to create a dashboard that allows filtering of:**  
1.👨‍💻 Job Title  
2.🌏 Job Country  
3.💹 Job Type
